A mobile-first comic reader for WordPress with swipe navigation and responsive design.

== Description ==

SwipeComic is a modern, mobile-first comic reader plugin for WordPress. It provides an intuitive swipe-based navigation system optimized for touch devices while maintaining full desktop compatibility.

**Features:**

* Mobile-first responsive design
* Touch-optimized swipe navigation
* Lightweight React-based implementation
* Modern build system with asset optimization
* Follows WordPress Coding Standards
* Internationalization ready

**Usage:**

Create comic episodes using the SwipeComic custom post type in the WordPress admin. Episodes are displayed using custom templates and organized into series.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/swipecomic/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure settings under Settings > SwipeComic
4. Create comic episodes using the SwipeComic post type
5. Organize episodes into series using the Series taxonomy

== Frequently Asked Questions ==

= How do I create a comic episode? =

Go to SwipeComic > Add New in your WordPress admin, upload images, and configure episode settings.

= Is this plugin mobile-friendly? =

Yes! SwipeComic is built with a mobile-first approach and includes touch-optimized swipe navigation.
